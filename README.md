ToDo / Note taking app with dark theme 

## Fatures
~ Add new ToDo / Note <br>
~ Edit active item <br>
~ Swipe active item to dismiss it <br>
~ Swipe done item back to active <br>
~ Widget for active ToDos <br>
~ Widget for selected Note <br>
